QuickQWERTY 1.0-beta3

Release date: 2013-09-29

QuickQWERTY 1.0-beta3 is a beta release during the development of QuickQWERTY 1.0. This section lists all new features to be available in QuickQWERTY 1.0.

  • New: Tutor with 21 units and 101 lessons to learn touch typing.
  • New: Facebook like box in the sidebar.
  • New: Error message is displayed when JavaScript is not enabled in the browser.
  • New: Support for two styles of typing the number keys: 6–7 split and 5–6 split.
  • New: QuickQWERTY version number in the footer.
  • New: Usage details logging via http://susam.in/onepixel/?js.
  • New: Input commands: reset, rst, fix, xxx.
  • New: Version update notification is displayed when the user is using a version of QuickQWERTY that is older than the latest stable version.
