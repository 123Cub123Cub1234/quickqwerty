You are viewing an offline version of QuickQWERTY download page from your computer or from a web server other than quickqwerty.com. The download details mentioned in this page may be outdated.

QuickQWERTY can be used offline by downloading a zip or tar.gz file, extracting it on your computer, and then opening the index.html file in the extract. The latest development release is the recommended release for offline use.

[Release date: 2013-08-26]

This is a beta release. This release is currently undergoing real world testing. This release may be followed by more beta releases to fix issues found in this release. If no major issues are found in this release, then this release will be followed by a release candidate.

This release is not considered stable enough for regular usage by all users. This release is recommended only to those users who want to use the new features that are present in this release but not available in the stable release. The user must be aware that this release is unstable, and therefore it may contain issues.

Online release

▶ Use QuickQWERTY online

The online release of QuickQWERTY can always be accessed at http://quickqwerty.com/.

The online release is usually a snapshot of a recent development release, or the latest stable release if no development release has been made since the last stable release. Occasionally, the online release may contain features and fixes that will be available in a future development release.
